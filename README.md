# Python-OCR  
## 概要  
### OCRの学習用リポジトリ  
今回は仕事で使う個人的なOCRの学習用リポジトリを作成します。  
いわゆる備忘録的なものです。  
使用するもの  
- Python  
- EasyOCR  
- Tkinter  
  
### 使い方  
``` bash
pip install easyocr
easyocr_gui.py  
```  
で起動します。  
実行環境はWindows 11です。  
課題として処理が圧倒的に遅いので、GPUなどを使って高速化したいですね。  
また、読み取りの前処理として、画像のリサイズやグレースケール化なども行いたいです。  
以上YAMAでした。
